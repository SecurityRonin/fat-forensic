# Privacy Policy

`fat-core` and `fat-forensic` are offline Rust libraries. They run entirely on the
machine that invokes them.

- **No data collection.** The crates do not collect, transmit, or phone home any
  data. They have no network code.
- **No telemetry.** There is no analytics, crash reporting, or usage tracking.
- **Local processing only.** Input images are read from the local filesystem or
  an in-memory reader supplied by the caller; nothing leaves the host.

Because the software performs no data collection, there is no personal data for us
to store, share, or sell.
